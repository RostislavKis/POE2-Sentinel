# 🛡️ POE2 Sentinel

Многофункциональный вспомогательный инструмент для Path of Exile 2 с авто-ботом для фласок, оверлеем местности и функциями раскрытия карты.

> 🌐 Languages: [English](README.md) · **Русский**

## Возможности

- **Бот для фласок (Flask Bot)** — автоматически использует фласки, когда HP/Мана падают ниже порога
  - **Режим структур (Structure Mode)** (рекомендуется) — автоматически подстраивается под игровые патчи!
  - **Режим памяти (Memory Mode)** — быстрое чтение по цепочке указателей
  - **Режим OCR (OCR Mode)** — резервный вариант через захват экрана
- **Оверлей местности (Terrain Overlay)** — прозрачный радар, показывающий планировку карты, монстров, NPC и сундуки
- **Раскрытие тумана Атласа (Atlas Fog Reveal)** — переключение тумана войны на карте
- **Раскрытие через шейдер (Shader Reveal)** — постоянная видимость миникарты (требуется перезапуск игры)
- **Авто-обновления (Auto Updates)** — автоматическая проверка и установка новых версий

## Требования

- Windows 10/11 (64-бит)
- Path of Exile 2 (версия из Steam)
- **Запуск от имени Администратора** (требуется для чтения памяти)
- .NET Runtime (только для функции Shader Reveal)

## Быстрый старт

1. Запустите POE2 и войдите в игру
2. Запустите `POE2Sentinel.exe` от имени Администратора
3. Выберите режим детекции (рекомендуется Structure)
4. Настройте пороги по необходимости
5. Нажмите **START**

## Горячие клавиши

| Клавиша | Функция |
|---------|---------|
| F9  | Вкл./выкл. бота для фласок |
| F10 | Вкл./выкл. оверлей местности |
| F11 | Вкл./выкл. раскрытие тумана Атласа |

*Горячие клавиши можно переназначить в Настройках*

## Режимы детекции

| Режим | Описание |
|-------|----------|
| **Structure** | Использует AOB-сканирование — переживает игровые патчи! |
| Memory | Быстрые цепочки указателей — могут сломаться после патчей |
| OCR | Резервный захват экрана — работает всегда |

## Сборка из исходного кода

## Конфигурация

Настройки хранятся в `config.json`. Скопируйте `config.example.json`, чтобы начать.

Ключевые настройки:
- `detection_mode`: «structure», «memory» или «ocr»
- `life.threshold_percent`: порог HP для срабатывания фласки
- `life.pool_type`: «hp», «es» или «combined» (для билдов на энергощите)
